---
title: ubuntu 桌面2个显示屏分别显示自己的背景图
---

use [`HydraPaper`](!https://flathub.org/apps/details/org.gabmus.hydrapaper) Software to let each monitor show its own wallpaper

use flatpak to install HydraPaper
```bash
$ sudo apt install flatpak 
```

How to install and uninstall [HydraPaper](!https://ubunlog.com/en/hydrapaper-fondos-cada-monitor/) follow this article.
